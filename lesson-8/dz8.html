  2. Переменная а не определена в глобальном контексте.
    Есть функция с таким именем. Но вызвать ее можно, только изнутри.
                    var b = function a(x) {
                    x && a(--x);
                };
                alert(a);
  3. Возвращается функция. Т.к. код разбирается в 2 этапа (инициализация и интерпритация)
    При инициализации создаются переменные (var a). Переменная a = undefined
    При интерпритации объявление переменных игнорируется. Переменная a = функция. 
                    function a(x){
                    return x * 2;
                }
                var a;
                alert(a);
